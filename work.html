PyChess

PyChess is a chess game in Python I made which allows players to play against an AI or each other. I created this to learn more about the algorithms involved in these kind of games. The AI uses a Minimax algorithm to get the best move, I also used alpha-beta pruning to improve time performance. The decision function uses both piece-values and piece-tables to get the best move, this gives the AI a more "human" feel. For the GUI I used PyGame.

Discogs Cover Search

This is a project I created to help a friend who was transferring his record collection from vinyl to digital. It first reads headers from a csv file to create search params and confirms this with the user. Then is uses OAuth to create a connection to Discogs API. It then uses these params to search the Discogs database for corresponding covers and saves these.

Temperature Monitor

This is simple application to monitor system device temperatures on Linux systems. I used PyQt5 for the GUI. It allows users to log temperatures to a csv file, enable/disable devices and set the refresh rate. This simple applicatiton could easily be built upon to throttle fan controls, monitor temps from an externel source, etc.
